# Azeroth_Faker

An Azeorth Faker gem that provides data from the Warcraft universe

#Installing:
  gem install azeroth_faker

#Usage:
``` ruby
  AZEROTHFaker::Character.random_name #=>"Illidan Stormrage"

  AZEROTHFaker::Race.name #=> "Orc"
```
